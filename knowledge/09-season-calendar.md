## Fall Harvest Planning

### Calculating Fall Planting Dates
```
fall_sow_date = first_frost_date - days_to_maturity - 14 (fall factor)

# The "fall factor" adds 14 days because:
# - Days shorten in fall (slower growth)
# - Cooler temperatures slow maturity
# Some sources use 10-21 days depending on crop

# Example for Zone 4b (first frost Oct 1):
# Kale (55 days): Oct 1 - 55 - 14 = July 23 latest sow date
# Carrots (70 days): Oct 1 - 70 - 14 = July 8 latest sow date
# Beets (55 days): Oct 1 - 55 - 14 = July 23 latest sow date
```

## Month-by-Month Garden Calendar (Zone 4b)

### January
- [ ] Review last year's garden journal/notes
- [ ] Order seeds from catalogs
- [ ] Plan garden layout and crop rotation
- [ ] Start onion and leek seeds indoors (late January)

### February
- [ ] Continue starting onions, leeks, celery indoors
- [ ] Inventory seed starting supplies (trays, lights, soil mix)
- [ ] Submit soil test if not done in fall
- [ ] Review and update garden plan

### March
- [ ] Start brassicas indoors (early-mid): broccoli, cabbage, cauliflower
- [ ] Start peppers and eggplant indoors (early-mid)
- [ ] Start head lettuce indoors
- [ ] Begin hardening off early seedlings late month
- [ ] Prune fruit trees before bud break

### April
- [ ] Start tomatoes indoors (mid-April)
- [ ] Start basil indoors
- [ ] Direct sow outdoors (mid-April): peas, spinach, lettuce, radishes
- [ ] Plant potato seed pieces
- [ ] Transplant onion starts outdoors
- [ ] Transplant brassica seedlings outdoors (with frost protection ready)
- [ ] Prepare beds: amend soil, add compost
- [ ] Set up trellises, cages, supports
- [ ] Plant asparagus and rhubarb crowns

### May
- [ ] Transplant tomatoes, peppers, eggplant AFTER May 10-15
- [ ] Direct sow beans, corn, cucumbers, squash after May 15
- [ ] Direct sow beets, carrots, kohlrabi
- [ ] Plant sweet potato slips (late May)
- [ ] Succession sow lettuce, radishes
- [ ] Mulch established beds
- [ ] Install drip irrigation
- [ ] Begin regular watering schedule

### June
- [ ] Succession sow beans
- [ ] Start fall brassica transplants indoors
- [ ] Transplant fall brassicas outdoors (late June)
- [ ] Side-dress heavy feeders with compost/fertilizer
- [ ] Scout for pests weekly
- [ ] Weed consistently
- [ ] Begin harvesting: lettuce, spinach, radishes, peas

### July
- [ ] Direct sow fall crops: beets, carrots, kale, turnips
- [ ] Succession sow lettuce, beans
- [ ] Harvest: beans, cucumbers, summer squash, tomatoes begin
- [ ] Water consistently (1-1.5" per week)
- [ ] Monitor for disease; remove affected foliage
- [ ] Pinch tomato suckers if desired

### August
- [ ] Direct sow: last succession lettuce, radishes, spinach
- [ ] Peak harvest: tomatoes, peppers, corn, cucumbers, beans
- [ ] Process/preserve harvest (can, freeze, dry)
- [ ] Remove spent crops; plant cover crops in empty beds
- [ ] Scout for late-season pests

### September
- [ ] Harvest winter squash, pumpkins when stems dry
- [ ] Plant garlic cloves (late September)
- [ ] Protect tender crops from early frosts (row covers)
- [ ] Continue harvesting cold-hardy crops
- [ ] Clean up finished beds
- [ ] Plant cover crops (winter rye, crimson clover)
- [ ] Divide perennials

### October
- [ ] Final harvest of all remaining crops before hard freeze
- [ ] Pull and compost or dispose of plant debris
- [ ] Mulch garlic bed with 4-6" straw
- [ ] Winterize irrigation system
- [ ] Clean and store tools, cages, stakes
- [ ] Submit fall soil test
- [ ] Plant spring-flowering bulbs
- [ ] Cover beds with leaves/mulch or cover crop

### November
- [ ] Harvest from cold frames/row covers if applicable
- [ ] Review season: what worked, what didn't
- [ ] Update garden journal
- [ ] Begin next year's planning

## Garlic Calendar (Special — Fall Planted)

| Task              | Timing               | Notes                              |
|-------------------|---------------------|------------------------------------|
| Plant cloves      | Late Sept-early Oct  | 4-6 weeks before ground freeze     |
| Mulch heavily     | After planting       | 4-6" straw; insulates through winter |
| Spring emergence  | April                | Remove excess mulch when shoots appear |
| Scape removal     | June                 | Cut scapes for bigger bulbs (hardneck)|
| Harvest           | Mid-July             | When bottom 3-4 leaves brown       |
| Cure              | 2-4 weeks            | Hang in dry, shaded, ventilated area |
| Storage           | After curing         | Cool (50-60°F), dry, dark; 6-8 months |

## Key Formulas for Season Planning

```
# Days available for a crop
available_days = first_frost_date - transplant_date
# Crop fits if: days_to_maturity <= available_days

# Indoor seed start date
start_date = transplant_date - (weeks_before_transplant * 7)

# Fall planting last sow date
last_sow = first_frost_date - days_to_maturity - fall_factor_days
# fall_factor_days: 14 for most crops, 7 for fast growers, 21 for slow

# Succession planting count
num_sowings = floor((last_sow_date - first_sow_date) / succession_interval_days) + 1

# Season extension impact
# Row cover: extends ~2-4 weeks each end
# Cold frame: extends ~4-6 weeks each end
# High tunnel: extends ~6-8 weeks each end
# Effective growing season = base_season + (2 * extension_weeks)
```
